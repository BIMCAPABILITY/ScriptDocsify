# GEN_3D: ISOLATE ELEMENTS BY WARNINGS

> #### DESCRIPTION: 
- **Isolates all clashing geometries (by categories) in the current active view and creates a new 3D View with clashing elements.**
- **e.g. walls versus pipes**
